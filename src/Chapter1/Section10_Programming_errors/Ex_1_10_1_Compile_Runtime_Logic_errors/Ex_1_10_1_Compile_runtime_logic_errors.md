1.10.1 What are syntax errors (compile errors), runtime errors, and logic errors?
---

Syntax errors are detected by compilers due to errors in code construction, such as mistyping a keyword,
omitting some necessary punctuation, or using an opening brace without a corresponding closing brace.

Runtime errors occur during execution of the program and cause a program to terminate abnormally.
They occur while a program is running if the environment detects an operation that is impossible to carry out.
(e.g. division by 0, the program asks for an integer, but the user enters a String)

Logic errors results in incorrect results, when a program does not perform the way it was intended to.
(e.g. results is truncated when we expected a float result).
