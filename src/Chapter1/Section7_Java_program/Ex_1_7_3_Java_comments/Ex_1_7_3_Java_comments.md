1.7.3 What is a comment? Is the comment ignored by the compiler? How do you denote a comment line and a comment paragraph?
---

Comments are used to document what a program is for and how a program is constructed. 
Comments help the programmers or users to communicate and understand the program. 
Comments are not programming statements and are ignored by the compiler. 
In Java, comments are preceded by two forward slashes (//) in a line or 
enclosed between /* and */ in multiple lines. 

When the compiler sees //, it ignores all text after // in the same line. 
When it sees /*, it scans for the next */ and ignores any text between /* and */.
