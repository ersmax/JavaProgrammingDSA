1.9.1 Reformat the following program according to the programming style and documentation guidelines. Use the end-of-line brace style
---
```Java
public class Test
{
    // Main method
    public static void main(String[] args) {
    /** Display output */
    System.out.println("Welcome to Java");
    }  
}
```
---
This is the reformatting:
```Java
/** This class serves for testing purposes */
public class Test
{
    /** The main method signals the start of the program
     * @param args allows command-line arguments to be passed to the program
     * */
    public static void main(String[] args) {
        // Display output
        System.out.println("Welcome to Java");
    }  
}
```
