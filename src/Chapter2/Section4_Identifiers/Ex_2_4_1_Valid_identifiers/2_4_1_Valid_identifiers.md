2.4.1 Which of the following identifiers are valid? Which are Java keywords?
-
miles, Test, a++, ––a, 4#R, $4, #44, apps
class, public, int, x, y, radius

Valid:
- miles
- Test
- $4
- apps
- x
- y
- radius

Not valid:
- a++
- --a
- 4#R
- #44
- class
- public
- int

Keywords:
- class
- public
- int
